# Yard Sale
El proyecto consiste en un e-comerce, se creó en el **Curso Práctico de Frontend Developer** de Platzi. Dentro del directorio /original-components se encuentran todos los archivos (separados por clase) que contienen el código HTML y CSS de cada componente.  

Luego de hacer **fork** al repositorio original, el proyecto quedará completamente integrado con **Javascript**. Los componentes se encuentran disponibles en un mismo HTML, y con Javascript se pueden desplegar diferentes menu (mobile menu - desktop menu) y otros componentes.  

[Ver página web Yard Sale](https://federicogomez00.github.io/platzi-html_css_js/)
